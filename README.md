Platform DevUX is a Project where local dev enviornment build automation is maintained.

## Requirements

### Required software

* Git
* PowerShell

This project has been tested with the following versions of required software

    Git v2.39.2
    PowerShell 2023.2.1

### Required access

No privileged access required
